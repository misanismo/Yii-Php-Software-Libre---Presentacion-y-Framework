Historia PHP

  • Fue creado por Rasmus Lerdorf en 1994.

  • Originalmente fue escrito en C.

  • En junio de 1995, Rasmus publicó el código fuente de php, lo que permitió la solución de errores del código por medio de los usuarios.

    • PHP

  • Rasmus Lerdorf y Usuarios o comunidad

  • Sigue siendo uno de los lenguajes más usados en desarrollo web.

    • Aspectos de PHP

  • Lenguaje totalmente libre y abierto

  • Comunidad muy grande

  • Fácil acceso a bases de datos

  • Curva de aprendizaje muy baja

    • Ventajas de PHP

  • Capacidad de conexión con la mayoría de los motores de base de datos.
  • Es un lenguaje multiplataforma.


    • Desventajas de PHP

  • Si no lo configuras correctamente dejas abiertas muchas brechas de seguridad.
  • Se necesita instalar un servidor web.

    • Historia Yii

  • Creado por Qiang Xuw.

  • Fecha de lanzamiento: Diciembre del 2008.

  • Llegó a remplazar a PRADO como framework de PHP.

  • Licencia BSD (Berkeley Software Distribution)

    • Aspectos de Yii

  • Patrón de diseño MVC

  • Es utilzado para todo tipo de aplicaciones Web

    • Diferenciadores de Yii

  • Puede crear modelos de plantilla (MVC)

  • Es utilzado para todo tipo de aplicaciones Web

  • Manejo de formularios (Validación de campos).

    • Ventajas de Yii

  • Integración con jQuery.
  • Entradas de Formulario y validación.
  • Personalización de aspectos y temas.

    • Desventajas de Yii

  • Tiempo de aprendizaje.
  • Dependencia. Si lo escogemos, es imposible migrar a otro Framwork.
